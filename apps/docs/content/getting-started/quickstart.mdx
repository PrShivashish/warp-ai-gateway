# Quickstart

Get your first Warp API call working in under 2 minutes.

## Prerequisites

- A Warp account with an API key
- [Node.js](https://nodejs.org) 18+ or [Bun](https://bun.sh) installed

## Option 1: Using the SDK

Install the SDK:

```bash
npm install warp-sdk
```

Create a file called `hello.ts`:

```ts
import { Warp } from "warp-sdk";

const client = new Warp({
  apiKey: "sk-wp-your-api-key",
  baseUrl: "http://localhost:4000",
});

const response = await client.chat({
  model: "google/gemini-2.0-flash",
  messages: [{ role: "user", content: "Say hello in 3 languages" }],
});

console.log(response.completions.choices[0].message.content);
```

Run it:

```bash
npx tsx hello.ts
```

## Option 2: Using curl

```bash
curl http://localhost:4000/api/v1/chat/completions \
  -H "Authorization: Bearer sk-wp-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "google/gemini-2.0-flash",
    "messages": [{"role": "user", "content": "Say hello in 3 languages"}],
    "stream": false
  }'
```
