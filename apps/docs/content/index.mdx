# Welcome to Warp

Warp is a **unified LLM gateway** that gives you a single API to access multiple AI model providers.

## Why Warp?

- **One API, many models** — Switch between GPT, Gemini, Claude, and more with a single endpoint.
- **Provider abstraction** — Never deal with provider-specific APIs again.
- **Smart routing** — Requests are routed to the best available provider automatically.
- **Wallet billing** — Simple pay-as-you-go billing with real-time balance tracking.
- **Streaming support** — First-class SSE streaming for real-time token delivery.
- **TypeScript SDK** — Strongly-typed client for fast developer onboarding.

## Architecture

```
┌─────────────┐     ┌──────────────────┐     ┌──────────────┐
│  Your App   │────▶│    Warp API      │────▶│  Providers   │
│  (SDK/HTTP) │◀────│  /api/v1/chat/.. │◀────│  (OpenAI,    │
└─────────────┘     └──────────────────┘     │   Gemini...) │
                                              └──────────────┘
```

## Quick Example

```ts
import { Warp } from "warp-sdk";

const client = new Warp({ apiKey: "sk-wp-..." });

const response = await client.chat({
  model: "google/gemini-2.0-flash",
  messages: [{ role: "user", content: "Hello!" }],
});

console.log(response.completions.choices[0].message.content);
```
