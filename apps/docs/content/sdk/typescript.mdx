# TypeScript SDK

The official TypeScript SDK for Warp. Zero dependencies, strongly typed, and fast.

## Installation

```bash
npm install warp-sdk
# or
bun add warp-sdk
```

## Initialization

```ts
import { Warp } from "warp-sdk";

const client = new Warp({
  apiKey: "sk-wp-your-api-key",
  baseUrl: "http://localhost:4000", // optional, defaults to localhost:4000
  defaultModel: "google/gemini-2.0-flash", // optional
});
```

## Methods

### `chat(request)`

Send a non-streaming chat completion request.

```ts
const response = await client.chat({
  model: "google/gemini-2.0-flash",
  messages: [
    { role: "system", content: "You are a helpful assistant." },
    { role: "user", content: "What is the speed of light?" },
  ],
});

console.log(response.completions.choices[0].message.content);
console.log(
  `Tokens: ${response.inputTokensConsumed} in, ${response.outputTokensConsumed} out`,
);
```

### `chatStream(request)`

Send a streaming request. Returns an `AsyncGenerator<string>` that yields tokens.

```ts
for await (const token of client.chatStream({
  messages: [{ role: "user", content: "Write a haiku about coding" }],
})) {
  process.stdout.write(token);
}
console.log(); // newline
```

### `models()`

List all available models.

```ts
const models = await client.models();

for (const model of models) {
  console.log(`${model.slug} — ${model.company.name}`);
}
```

## Error Handling

The SDK throws `WarpError` for API errors:

```ts
import { Warp, WarpError } from "warp-sdk";

try {
  await client.chat({ messages: [{ role: "user", content: "Hi" }] });
} catch (err) {
  if (err instanceof WarpError) {
    console.error(`HTTP ${err.status}: ${err.message}`);
    // err.body contains the raw response body
  }
}
```

## Types

All types are exported for use in your application:

```ts
import type {
  Message,
  ChatRequest,
  ChatResponse,
  Model,
  WarpConfig,
} from "warp-sdk";
```

## Advanced: SSE Parser

The internal SSE parser is exported for custom streaming implementations:

```ts
import { parseSSEStream } from "warp-sdk";

const response = await fetch("http://localhost:4000/api/v1/chat/completions", {
  /* ... */
});

for await (const token of parseSSEStream(response)) {
  // handle token
}
```
