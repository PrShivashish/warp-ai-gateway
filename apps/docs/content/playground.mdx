# Developer Playground

Test Warp models instantly in your browser. No SDK installation required.

<Playground />

## How it works

1. **Enter your API Key**: Your key is only stored in your browser's memory and is sent directly to our API.
2. **Select a Model**: Choose from 200+ models available on Warp.
3. **Start Chatting**: Send messages and see streaming responses in real-time.

> [!TIP]
> You can find your API keys in the [Dashboard](http://localhost:3001/api-keys).
