# Assets Directory

Place your background video file here.

## Video File

- **File name**: `background-video.mp4` (or `.webm`)
- **Location**: `/public/assets/background-video.mp4`
- **Recommended format**: MP4 (H.264 codec) or WebM for better browser compatibility
- **Recommended size**: Optimize for web (consider using a compressed version to reduce file size)

The video will be used as a fixed background on all dashboard pages with low opacity (15% by default).
